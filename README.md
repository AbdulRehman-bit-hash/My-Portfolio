# Abdul Rehman — Portfolio Website

> Personal portfolio of Abdul Rehman, a Full Stack Developer (MERN Stack) based in Lahore, Pakistan.

---

## 🌐 Live Preview

Open `abdulrehman_portfolio.html` directly in any modern browser — no build step or server required.

---

## 📁 Project Structure

```
portfolio/
└── abdulrehman_portfolio.html   # Single-file portfolio (HTML + CSS + JS)
```

Everything — styles, layout, and interactivity — lives in one self-contained HTML file.

---

## ✨ Features

- **Fully responsive** single-page layout
- **Smooth scroll** navigation with a fixed navbar
- **Fade-in animations** triggered on scroll (IntersectionObserver)
- **5 sections** — Hero, About, Skills, Projects, Achievements, Contact
- **Zero dependencies** — no frameworks, no npm, no build tools
- Google Fonts loaded via CDN (Syne · DM Sans · DM Mono)
